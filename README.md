Sistema de biblioteca básico feito em colaboração por Gustavo Breder, Gabriel Rosa e Pedro Luiz.

No código você pode:

- Cadastrar livros 
- Pedir empréstimo de livros
- Devolver livros emprestados
- Listar livros cadastrados 
- Salvar a lista de livros cadastrados e o status
- Carregar a lista de livros salva anteriormente
